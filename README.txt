PAYCHECK PLANNER WEB APP

1. Open index.html in a modern browser.
2. The app saves your bills, paychecks, goals, and settings in the browser on that device.
3. To use it like an app on iPhone, host the folder on a website and use Safari > Share > Add to Home Screen.
4. Reset Data restores the original spreadsheet values.

Included features:
- Dashboard
- Unlimited bills, paychecks, and goals
- Editable paycheck amounts and extra income
- Rent and other split bills divided across all paychecks in the selected month
- Non-split bills assigned to the paycheck before their due date
- Monthly goal contributions divided across paychecks
- Leftover cash target and minimum savings alert
- Device-local data storage
