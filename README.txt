PAYDAY COMBINED MODERN

GitHub Pages update:
1. Unzip this package.
2. Upload every file and the assets folder directly to the root of the repository.
3. Replace the old files when GitHub asks.
4. Commit changes.
5. Wait for the Pages deployment green check.
6. Open the site and refresh. If an older version remains, remove and re-add the Home Screen app or clear Safari website data for the site.

Data is stored locally in the browser on each device. Use Settings > Export Backup before replacing or clearing browser data.
