PAYDAY PAYCHECK-FIRST PLANNER v4.1

Upload every file and the assets folder directly to the root of your GitHub Pages repository.
Replace the previous files, commit, wait for the green Pages deployment check, then fully close and reopen the iPhone Home Screen app.

Core behavior:
- Set one next-check date, pay frequency, and standard base paycheck.
- The app generates a rolling 12-month schedule and rolls the next check forward automatically.
- Add multiple overtime, reimbursement, bonus, side-income, or other entries to individual checks.
- Bills use recurring due days only. No bill month is required.
- Split bills support a first-check percentage; the next eligible check receives the remainder.
- Data is stored locally in the browser and can be exported/imported.
